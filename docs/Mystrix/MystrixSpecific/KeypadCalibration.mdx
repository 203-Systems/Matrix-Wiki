# Keypad Calibration

:::danger
This feature is only for power user and not recommended for general user. 

If the keypad is incorrectly calibrated, your device will not work as expected. In the worst case, you may need to reflash the OS and wipe all the data to recover the device.
:::

:::info
This feature is only available for Mystrix with force sensitive keypad, for example, Mystrix Pro.
:::

Keypad Calibration APP is specific to Mystrix with force sensitive keypad. The app is used to calibrate the keypad to the user's preference. You can adjust the minimal trigger force and the maximal trigger force of the keypad.

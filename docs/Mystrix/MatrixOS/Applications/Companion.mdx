# Companion App

Companion app is used to connect with Bitfocus Companion to enable advanced and very power control to various things.

Mystrix intergration is available after the 3.4.0 release of the Bitfocus Companion.

Download the latest version of Bitfocus Companion from [here](https://bitfocus.io/companion/)

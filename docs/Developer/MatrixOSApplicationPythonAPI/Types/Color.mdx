# Color

## Preview notice
The MatrixOS Python API is in preview and is subject to change; it may contain errors.


## Overview

The `Color` class represents RGB color values for LED control. Colors can be created using individual RGB values or hexadecimal notation.

*The Color class is implemented in <GithubLink path="Applications/Python/PikaPython/MatrixOS_Color.py"/> with type hints in <GithubLink path="Applications/Python/PikaPython/_MatrixOS_Color.pyi"/>.*

---

## `Color(*val)`
```python
class Color:
    def __init__(self, *val) -> None
```

Creates a color using various input formats:
- **0 arguments**: Creates black color (0, 0, 0)
- **1 argument**: WRGB hexadecimal color value
- **3 arguments**: R, G, B values (0-255)
- **4 arguments**: R, G, B, W values (0-255)

**Parameters:**
- `*val`: Variable arguments depending on desired constructor

## Static Factory Methods

### `FromRGB`
```python
def FromRGB(r: int, g: int, b: int) -> Color
```

Creates a color from RGB values.

**Parameters:**
- `r` (`int`): Red component (0-255)
- `g` (`int`): Green component (0-255)
- `b` (`int`): Blue component (0-255)

**Returns:**
- `Color`: New color instance


### `FromRGBW`
```python
def FromRGBW(r: int, g: int, b: int, w: int) -> Color
```

Creates a color from RGBW values.

**Parameters:**
- `r` (`int`): Red component (0-255)
- `g` (`int`): Green component (0-255)
- `b` (`int`): Blue component (0-255)
- `w` (`int`): White component (0-255)

**Returns:**
- `Color`: New color instance


### `FromHex`
```python
def FromHex(wrgb: int) -> Color
```

Creates a color from a hexadecimal WRGB value.

**Parameters:**
- `wrgb` (`int`): Hexadecimal color value

**Returns:**
- `Color`: New color instance

## Color Manipulation Methods

### `Scale`
```python
def Scale(self, factor: float) -> Color
```

Scales the color intensity by a factor.

**Parameters:**
- `factor` (`float`): Scaling factor (0.0 to 1.0+)

**Returns:**
- `Color`: New scaled color


### `Dim`
```python
def Dim(self, factor: float) -> Color
```

Dims the color by a factor.

**Parameters:**
- `factor` (`float`): Dimming factor (0.0 to 1.0)

**Returns:**
- `Color`: New dimmed color


### `DimIfNot`
```python
def DimIfNot(self, not_dim: bool, factor: float) -> Color
```

Conditionally dims the color based on a boolean condition.

**Parameters:**
- `not_dim` (`bool`): If True, don't dim; if False, apply dimming
- `factor` (`float`): Dimming factor (0.0 to 1.0)

**Returns:**
- `Color`: Original or dimmed color based on condition

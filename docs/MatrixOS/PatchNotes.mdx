## Matrix OS 3.0 - Go Make Some Noise, Go Make It Yours
*Released: November 7th, 2025*

[Download (GitHub)](https://github.com/203-Systems/MatrixOS/releases/tag/V3.0.0)

Matrix OS 3.0 brings long awaited Python application support, file system capabilities, and super enhanced Note App with powerful new musical features.

### New Features
- **Python Application Support** : Write your own apps directly in Python
- **Python REPL**: Interactive Python shell for real-time coding and experimenting.
- **Matrix OS File System API**: File system for applications to use
- **Micro SD Support**: Load APP from Micro SD cards
- **USB Mass Storage Mode**: Access your SD card from your computer
- **Note App V2**: Complete rewrite with modern interface, new layouts, and expressive features
- **Note App V2 - Control Bar**: New controls bar for pitchbend, transpose control, octave, swap layout, and set of features
- **Note App V2 - Note Latch**: Latch note with one toggle
- **Note App V2 - Chord Mode**: Perform chord using a key combo
- **Note App V2 - Arpeggiator**: Advanced arpeggiator with multiple modes and configurations
- **Note App V2 - Custom Scale**: Build your own scales
- **Note App V2 - Diatonic Scale**: Play in scale with modern diatonic options
- **Note App V2 - MIDI Clock Out**: Send MIDI clock sync to external hardware and DAWs
- **Lighting App BPM Sync**: Synchronize lighting effect to BPM for audio-visual scenario
- **Fast Text Scroll System**: Advance mode for faster text scroll, making you recall function quickly
- **Customizable Application Launcher**: Create sections, rearrange, and hide apps from your application launcher
- **C++ Application Repo Linking**: Matrix OS now can link and compile user app from remote repo by just adding a line in application list

### Improvements
- **Enhanced LED performance**: Reduced glitching and improved visual quality
- **Faster and smoother UI transitions**: Better user experience throughout the system
- **Improved system responsiveness**: Overall stability and performance enhancements
- **Numerous system-wide refinements**: Better reliability across all applications

### Bug Fixes
- **Resolved Note App display and control inconsistencies**: Better user interface behavior
- **General stability fixes**: Across all apps for improved reliability

### Changes
- **System-wide UI rework**: A lot of APP got their UI tweaked, most noticeable one is that brightness control and rotation control will be in system setting only now.
- **Poly Playground App removed**: Fully integrated into Note App V2

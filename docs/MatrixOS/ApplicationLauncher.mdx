# Application Launcher

:::warning
The application launcher layout is customizable. The view in the visualizer might not be exact as the view on your device.
:::

The Application Launcher is the main menu of Matrix OS. It allows users to select an application to use.

The top section of the Application Launcher shows the available applications. The bottom section features the app catalog selection on the left and <DocLink to="/docs/MatrixOS/SystemSettings">system settings</DocLink> on the right.

## Customize Layout

You can fully customize your application launcher layout to organize apps according to your preferences. The customization features include:

### Application Management
- **Select App**: Tap an app to select it (it will light up bright while others dim). Tap any empty space to deselect.
- **Move Apps**: With an app selected, tap another app to swap their positions.
- **Organize in Folders**: Select an app and tap a folder to move it into that folder
- **Hide Apps**: Remove apps from the launcher view that you don't frequently use

### Folder Management
- **Create Folders**: Empty folder slots will have a breathing animation - tap them to create a new folder
- **Customize Folders**: Hold on an existing folder to:
  - Change the folder color
  - Delete the folder (apps inside will return to the folder 1)
- **Organize Apps**: Move apps between folders to keep your launcher organized

# Application Launcher

The Application Launcher is the main menu of Matrix OS. It allows users to select an application to use.

The top section of the Application Launcher shows the available applications. The bottom section features the app catalog selection on the left and [system settings](/docs/MatrixOS/SystemSettings) on the right.

:::note
Currently, the stock application catalog is the only one available. In the future, more application catalogs will be added to the bottom left corner.
:::

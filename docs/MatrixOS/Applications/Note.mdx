# Note App

The Note APP is used as a MIDI note interface. It can be used to play MIDI notes with different layout, scale, and modes.

After launching, you will enter the [note pad](#note-pad), click the function key to enter the [config menu](#config-menu), click again to return to the [note pad](#note-pad).

To exit note app, enter [config menu](#config-menu) and hold down the function key in the [config menu](#config-menu). You will be brought back to the [application launcher](/docs/MatrixOS/ApplicationLauncher).

## Note Pad

Note pad is the main view of the note app. It shows the current note pad with the selected layout, scale, and mode. 

The pad will differ based the customization options, see [Example Layouts](#example-layouts) for some idea of how the note pad will look with different settings.

Clicking on the function key will bring you to the [config menu](#config-menu).

## Overlap
The Overlap Selector selects how much each row above overlaps with the row below it. This is useful if you want to play chords with one hand.

There is also an align root toggle inside this menu. When on the next row above after the entire scale is completed will realign with root note.

This setting is layout config independent.

## Spilt Mode

The Spilt Mode Selector allows you to enable or disable spilt mode. Spilt mode can be no spilt, horizontal spilt, or vertical spilt.


The Spilt Mode Selector will show different color based on the current spilt mode.

- No spiit mode: Dim White
- Vertical spilt mode: Cyan
- Horizontal spilt mode: Purple

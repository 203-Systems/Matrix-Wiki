# Note App

The Note App serves as a MIDI note interface, allowing users to play MIDI notes with various layouts, scales, and modes.

Upon launching, you will enter the [note pad](#note-pad). Click the function key to access the [config menu](#config-menu), and click again to return to the [note pad](#note-pad).

To exit the Note App, enter the [config menu](#config-menu) and hold down the function key. This will return you to the [application launcher](/docs/MatrixOS/ApplicationLauncher).

## Note Pad

The Note Pad is the main view of the Note App. It displays the current note pad with the selected layout, scale, and mode.

The appearance of the pad will vary depending on the customization options. Refer to [Example Layouts](#example-layouts) to see how the notepad looks with different settings.

Clicking the function key will bring you to the [config menu](#config-menu).

### Enforce Scale

When Enforce Scale is enabled, only notes in the selected scale will be shown. When disabled, all notes will be shown, but notes that are not in the selected scale will be dimmed.

## Layout Selector

The Layout Selector allows you to choose between different note pad layout modes:

- **Octave Layout**: Traditional octave-based layout
- **Offset Layout**: Customizable offset pattern with configurable X and Y offsets
- **Piano Layout**: Piano-style key arrangement

## Overlap

The Overlap Selector determines how much each row above overlaps with the row below it. This is useful for playing chords with one hand.

There is also an `Align Root` toggle within this menu. When enabled, the next row above, after the entire scale is completed, will realign with the root note.

This setting is independent of the layout configuration.

### Align Root

If the align root option is enabled, the next row above will realign with the root note after completing the entire scale. If the align root option is disabled, the layout will continue from the note where the last row ended.

## Split Mode

The Split Mode Selector allows you to enable or disable split mode. Split mode can be set to no split, horizontal split, or vertical split.

The Split Mode Selector will display different colors based on the current split mode:

- No split mode: Dim White
- Vertical split mode: Cyan
- Horizontal split mode: Purple

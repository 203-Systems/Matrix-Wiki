# Note App

The Note APP is used as a midi note interface. It can be used to play midi notes with different layout, scale, and modes.

After launching, you will enter the [play view](#play-view), click the function key to enter the [config menu](#config-menu), click again to return to the [play view](#play-view).

To exit note app, enter [config menu](#config-menu) and hold down the function key in the [config menu](#config-menu). You will be brought back to the [application launcher](/docs/MatrixOS/ApplicationLauncher).

# Note App

The Note App (V2) serves as a comprehensive MIDI note interface, allowing users to play MIDI notes with various layouts, scales, modes, and advanced features including auto chords, arpeggiators, split views, and customizable color presets.

Upon launching, you will enter the [note pad](#note-pad). Click the function key to access the [config menu](#config-menu), and click again to return to the [note pad](#note-pad).

To exit the Note App, enter the [config menu](#config-menu) and hold down the function key. This will return you to the [application launcher](/docs/MatrixOS/ApplicationLauncher).

## Note Pad

The Note Pad is the main view of the Note App. It displays the current note pad with the selected layout, scale, and mode.

The appearance of the pad will vary depending on the customization options. Refer to [Example Layouts](#example-layouts) to see how the note pad looks with different settings.

Clicking the function key will bring you to the [config menu](#config-menu).

## Layout Selector

The Layout Selector allows you to choose between different note pad layout modes:

- **Octave Layout**: Octave-based layout. Each row will start with the root note. If the scale has more than 8 notes, it will span 2 rows.
- **Offset Layout**: Customizable offset pattern with configurable X and Y offsets
- **Chromatic Layout**: Traditional note layout. Notes are place in order, left to right, bottom to top.
- **Piano Layout**: Piano style key arrangement

## Arpeggiator Settings

The Arpeggiator Setting Menu provides comprehensive control over the arpeggiator settings. This menu is only accessible when the control bar is enabled in the Play View.

The arpeggiator setting menu has 7 different parameter pages, each with a distinct color coding:

- **BPM** (Red): Controls the tempo/speed of the arpeggiator
- **Swing** (Orange): Adjusts the timing swing between notes
- **Gate** (Yellow): Sets the note duration/gate time
- **Direction** (Green): Selects the arpeggiator pattern direction
- **Step** (Cyan): Controls the step the arp will progress (Loop # times with step offset added each loop)
- **Step Offset** (Blue): Adds pitch offset notes after each step
- **Repeat** (Magenta): Sets how many times the pattern repeats

## Control Bar

The Control Bar provides quick access to various controls and effect when enabled in the Play View. 

Each button have different shift feature as well. Holding any of the octave button will enable shift mode.

Features:
- **Pitch Bend Down** (X=1): Velocity-mapped pitch bend down (Shift: MIDI Stop)
- **Pitch Bend Up** (X=2): Velocity-mapped pitch bend up (Shift: MIDI Play)
- **Note Latch** (X=3): Note sustain with three states (Shift: Enable Toggle Latch Mode)
- **Chord** (X=4): Open Chord Control
- **Arpeggiator** (X=5): Open Arpeggiator Speed Control (Shift: Enter Arpeggiator Setting)
- **Key Mode** (X=6): Change the key of current layout (Shift: Enter Scale Selector)
- **Octave Down & Up** (X=7 & 8): Transpose notes by octave (Hold any as Shift)
- **Swap Layout**: Tap Octave Down + Octave Up together to swap layout.

**Split View Behavior**: The control bar only affects the primary note pad (left pad in vertical split, top pad in horizontal split). However, any modes or effects you enable will persist when switching between pads. You can configure latch, chord, or arpeggiator on one pad, then switch to the secondary pad and those settings will continue to function independently.

:::note
**Non-Persistent**: Control bar configurations (latch, chord, arpeggiator, etc.) are not saved across sessions. When you exit and restart the Note App, all control bar settings will reset to their default states.
:::

### Octave Control / Shift

Octave controls allow you to transpose all notes up or down by octaves. These buttons also serve as the shift modifier for accessing secondary functions.

**Controls**:
- **Octave Down** (X=7): Decrease octave
- **Octave Up** (X=8): Increase octave

**Special Functions**:
- **Hold Either Button**: Activates shift mode for other controls
- **Tap Both Together**: Swap between primary and secondary layouts in split view

**Visual Feedback**:
  - **Color**: Primary Note Pad Color
  - **Gradient**: Note Pad Octave

### Pitch Bend

The pitch bend controls allow you to bend the pitch of playing notes up or down. The amount of bend is velocity-mapped, meaning pressing harder creates a larger pitch bend.

**Controls**:
- **Pitch Bend Down** (X=1): Bends pitch down (red button)
- **Pitch Bend Up** (X=2): Bends pitch up (green button)
- **Shift + Pitch Bend Down**: Send MIDI Stop command
- **Shift + Pitch Bend Up**: Send MIDI Play command

### Note Latch

Note Latch keeps the notes you pressed sustained after release. If you hold multiple keys, they will all be sustained. When a new note is pressed after you've released all previously latched notes, all notes will be freed and the new note will be latched.

**Controls**:
- **Tap** (X=3): Toggle Latch Enable
- **Shift Tap**: Enable Toggle Latch mode

**Color States**:
- **Yellow** (Default): Latch disabled - notes stop when released
- **White**: Latch enabled - notes sustain until cleared
- **Magenta**: Toggle mode - each press on note pad toggles note on/off

To clear all latched notes, turn off the note latch.

### Chord Control

Chord control transforms single note presses into full chords, allowing you to play complex harmonies with individual keys. The chords automatically adapt to your selected scale and root note.

**Controls Bar Button**:
- **Tap** (X=4, Y=8): Open chord selector interface
- **Hold**: Quick toggle chord mode on/off
- **Shift Tap**: Toggle chord combinations directly

**Color States**:
- **Cyan**: Chord control closed
- **White**: Chord control active

**Chord Building Control**:
- **Base**: Major, Minor, Diminished, or Suspended. You can select from one of them
- **Multiple extensions**: 6th, 7th, 9th - You can select multiple to combine freely
- **Hold Combo**: Hold a combo to chord while you perform on the note pad
- **Shift Tap**: Latch chord types and extensions
- **Inversion**: Select an inversion or slide across them when chord is latched to strum.

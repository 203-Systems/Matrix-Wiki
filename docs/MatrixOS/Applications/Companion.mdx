# Companion App

Companion app is used to connect with [Bitfocus Companion](https://bitfocus.io/companion/) to enable very advanced and power control to various things.

Mystrix intergration is available since the 3.4.0 release of the [Bitfocus Companion](https://bitfocus.io/companion/).

# Companion App

The Companion App is used to connect with [Bitfocus Companion](https://bitfocus.io/companion/) to enable advanced and powerful control over various devices.

:::info
Mystrix integration has been available since the 3.4.0 release of the [Bitfocus Companion](https://bitfocus.io/companion/).
:::

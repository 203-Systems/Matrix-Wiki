# Poly Playground App

Poly Playground is a simple app that's inspired by the Poly Playground App by Mike Gao [App Store](https://apps.apple.com/us/app/polyplayground/id511514938)8

:::note
This app is only available in the Matrix OS 2.6 beta version. It will be merged into Note App v2 in the future update - See [Note App V2](/docs/Backroom/NoteAppV2.mdx)
:::

## Note Map

The note pad is chromatic, which means every note is a semitone apart. On the X axis, each key is spaced by 3 semitones, and on the Y axis, each key is spaced by 4 semitones.

For pad with the same color, they are using the semitone (but migth be different octave).

# Gamepad App

The Gamepad App is a simple demonstration of the upcoming customizable keypad feature of Matrix OS. It will be replaced by the customizable keymap app in a future release.

:::tip
Currently, we have a fixed layout for the control view, but you will soon be able to build your own layout via the [Matrix OS Keymap Editor](https://edit.203.io).
:::

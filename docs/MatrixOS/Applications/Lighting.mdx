# Lighting App

Lighting App is used to make your device as a lighting panel that you can use in different enviorments

Holding down the function key in [config menu](#config-menu) will exit this app and bring you back to the [application launcher](/docs/MatrixOS/ApplicationLauncher).

## Lighting View
Lighting view is the main view of the lighting app. It shows the current lighting effect and color.

Clicking on the function key will bring you to the [config menu](#config-menu).

## RGB Mode
In RGB mode, you can set the color of the lighting view to any color you want via the [Color Picker](/docs/MatrixOS/SharedUI/ColorPicker).

## Temperature Mode
In Temperature mode, you can set the color of the lighting view to a color that represents a temperature. The color will change based on the temperature you set. 

The selector UI is similar to the [Color Picker](/docs/MatrixOS/SharedUI/ColorPicker), but the hue selector will be replaced with a temperature selector and there's no saturation selector.

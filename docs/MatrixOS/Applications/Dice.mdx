# Dice App

The Dice App is a simple app that allows you to roll a dice.

The dice has two modes: dotted dice and number dice.

You can customize the number of faces on the dice (2 to 9 for dot mode, 2 to 99 for number mode) and the color and effect of the animation.

Tap the function key to access the config menu from the dice view.

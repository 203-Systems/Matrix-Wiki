# Strum App

Strum is a chord-based musical instrument app inspired by devices like the Omnichord. You can select chords and strum them using the Mystrix's touchbars on the left and right sides of the device.

:::note Device Orientation Required
The Strum app requires the device to be in portrait mode (rotated left or right) to access the touchbars for strumming functionality.
:::

Tap the function key to access the action menu from the main interface.

## How to Use

### Getting Started
1. **Device Orientation**: Rotate your device to portrait mode (left or right) to access the touchbars
2. **Chord Selection**: Tap any chord button on the bottom row to select your desired chord type
3. **Octave Adjustment**: Use the octave shifter (row 2) to adjust the pitch range
4. **Duration Control**: Use the duration modifier (row 3) to set note length

### Strumming
- **Left Touchbar**: Touch and drag along the left side touchbar to strum the chord
- **Right Touchbar**: Touch and drag along the right side touchbar to strum the chord
- **Chord Visualization**: The top area shows the selected chord notes and root key

### Available Chord Types
- **Major**: Basic major triad (1-3-5)
- **Minor**: Basic minor triad (1-♭3-5)
- **Major 7th**: Major triad with major seventh (1-3-5-7)
- **Minor 7th**: Minor triad with minor seventh (1-♭3-5-♭7)
- **Augmented**: Augmented triad (1-3-#5)
- **Diminished**: Diminished triad (1-♭3-♭5)

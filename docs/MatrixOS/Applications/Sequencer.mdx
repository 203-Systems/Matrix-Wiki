# Sequencer App

A powerful sequencer for Matrix OS. Learning about the [Note App](/docs/MatrixOS/Applications/Note) first is highly recommended.

:::warning Preview version
The documention here is incomplete and might contain factual errors. Matrix OS 3.2 is still not in final release yet so things in the OS might change as well.
:::

## Core Concepts

- Track: An independent lane with its own clip, patterns, color, channel, and note layout. This sequencer have 8 tracks. 
- Clip: a container of patterns per track. Clips can be launched or switched. Each track in this sequencer have up to 14 clips.
- Pattern: a grid of steps. Patterns have a per-pattern length. Each clip in this sequencer have 16 patterns.
- Step: a time slice in a pattern. A step can hold multiple events. Each pattern in this sequencer can have up to 16 steps, but you can also set micro steps.

### Track Selector

- Tap a track to select it. Selecting a different track clears step selections and active notes.
- Tap the current track to mark it as "active track selected" (affects Control Bar behavior).
- Clear + tap a track clears all steps in the current clip for that track.

### Pattern Pad

- Default view is 8x2 (16 steps). Two-pattern mode expands to 8x4 (two patterns at once).
- Tap a step to select it; release to deselect it.
- Tap a step while notes are selected on the Note Pad to add or remove those notes on that step.
- Tap a step to preview its notes (note on); release to stop preview (note off).
- Clear + tap step clears events in that step.
- Copy mode:
  - Tap a step with events to set the copy source.
  - Tap another step to copy events into it (supports cross-pattern copy).
- Shift + tap a step with events opens step detial menu (only when not playing that track).

### Note Pad

- Used for note entry and preview.
- Layout modes: Scale, Chromatic, Piano, Drum.
- Velocity-sensitive input when supported and enabled.
- In Sequencer view, if steps are selected, tapping notes toggles those notes on the selected steps.
- If the sequencer is playing and record is enabled, note input is recorded directly.

### Shift Modifier
This is the view if you held down any of the octave key. (They act as shift when held)

- Hold Shift: Preview Pattern Selector
- Tap Both Shift: Toggle Enable Pattern Selector
- Shift + Step: [Step Detail Menu](#step-detail-menu)
- Shift + Play (x1 in control bar): Resume Playing
- Shift + Copy (x6 in control bar): Toggle two pattern view

### Pattern Selector

- Shown when Pattern Selector is active (shift held or latched).
- Tap a pattern slot to select or switch patterns (switching is disabled while playing).
- Tap the "add" slot to create a new pattern.
- Clear + tap pattern deletes it.
- Copy mode:
  - Tap a pattern to set the copy source.
  - Tap another pattern to copy into it.
- Hold a pattern to enter length adjustment mode, then tap a length (1-16) to set that pattern's length.

## Clip Launcher

A grid for you to select clips to play. Similar to Ableton Live's Session mode.

- Normal mode: 8 tracks across, 7 clips high. Clip page selects page 1 or 2.
- Wide clip mode: 4 tracks across with 2 columns per track.

Not Playing:
- Tap a clip to select it.
- Tap an empty slot to create a clip.
- Shift + clip will start play this clip only.
- Clear + clip will deletes it.
- Copy mode: select a source clip, then tap a destination to copy it.

While Playing: 
- Tap a clip to queue it.
- Tap an empty slot to queue stop track.
- Double tap a clip to launch the entire row.

## Advance Tips
- You can copy or share your sequence by copy the sequence file using the USB Storage Mode in [System Setting](/docs/MatrixOS/SystemSettings). The files are located `Drive:\MatrixOS\AppData\203 Systems-Sequencer\sequences\#\`
- There is also a prev folder for each sequence, they are your previously saved files. if you accidently over wrote your sequence, you can revert them by using those files.

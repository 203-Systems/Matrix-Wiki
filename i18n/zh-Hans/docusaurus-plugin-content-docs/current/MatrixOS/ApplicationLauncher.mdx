# 应用启动器

:::warning
应用启动器的布局可以自定义。可视化器里的视图可能和你设备上看到的不完全一样。
:::

应用启动器是 Matrix OS 的主菜单，用户可以在此选择要使用的应用。

应用启动器顶部显示可用应用，底部左侧为应用目录选择，右侧为<DocLink to="/docs/MatrixOS/SystemSettings">系统设置</DocLink>。

## 自定义布局

你可以自定义应用启动器的布局，按你的习惯来摆放应用。包括：

- **选择应用**：点击应用以选择它（它会变亮，其他应用变暗）。点击空白区域取消选择。
### 应用管理
- **移动应用**：选中一个应用后，点另一个应用就能交换位置。
- **放进文件夹**：选中应用后，点一个文件夹就能把它放进去。
- **隐藏应用**：把不常用的应用从启动器里隐藏起来。

### 文件夹管理
- **创建文件夹**：空的文件夹插槽会有呼吸动画，点一下就能新建文件夹。
- **自定义文件夹**：长按现有文件夹可：
  - 更改文件夹颜色
  - 删除文件夹（里面的应用会回到文件夹 1）
- **整理应用**：在不同文件夹之间移动应用，让启动器更整齐。

## Matrix OS 3.0 - 尽情创作，随心定制
*发布日期：2025年11月7日*

[下载 (GitHub)](https://github.com/203-Systems/MatrixOS/releases/tag/V3.0.0)

Matrix OS 3.0 带来了期待已久的 Python 应用支持、文件系统功能以及功能强大的全新音符应用，包含丰富的音乐功能。

### 新功能
- **Python 应用支持** 🐍：使用 Python 直接编写应用
- **Python REPL**：交互式 Python 外壳，用于实时编码和实验
- **Matrix OS 文件系统 API**：供应用使用的文件系统
- **Micro SD 支持**：从 Micro SD 卡加载应用
- **USB 大容量存储模式**：从计算机访问 SD 卡
- **音符应用 V2**：完全重写，具有现代化界面、新布局和丰富功能
- **音符应用 V2 - 控制栏**：新的控制栏，包含弯音、移调控制、八度、切换布局等功能
- **音符应用 V2 - 音符锁定**：一键切换锁定音符
- **音符应用 V2 - 和弦模式**：使用按键组合演奏和弦
- **音符应用 V2 - 琶音器**：具有多种模式和配置的高级琶音器
- **音符应用 V2 - 自定义音阶**：构建自己的音阶
- **音符应用 V2 - 自然音阶**：使用现代自然音阶选项在音阶内演奏
- **音符应用 V2 - MIDI 时钟输出**：向外部硬件和 DAW 发送 MIDI 时钟同步
- **灯光应用 BPM 同步**：将灯光效果同步到 BPM，用于视听场景
- **快速文本滚动系统**：高级模式实现更快的文本滚动，快速回忆功能
- **可自定义的应用启动器**：创建分区、重新排列和隐藏应用启动器中的应用
- **C++ 应用仓库链接**：Matrix OS 现在可以通过在应用列表中添加一行来链接和编译来自远程仓库的用户应用

### 改进
- **增强的 LED 性能**：减少闪烁并改善视觉质量
- **更快更流畅的 UI 过渡**：整个系统提供更好的用户体验
- **改进的系统响应性**：整体稳定性和性能增强
- **众多系统范围的优化**：所有应用提供更好的可靠性

### 错误修复
- **解决了音符应用显示和控制不一致问题**：更好的用户界面行为
- **通用稳定性修复**：所有应用的可靠性改进

### 变更
- **系统范围的 UI 重做**：许多应用的 UI 进行了调整，最明显的是亮度和旋转控制现在仅在系统设置中
- **Poly Playground 应用已移除**：完全集成到音符应用 V2 中

## Matrix OS 2.6 - 掌控更多，自定更多，玩得更多
*发布日期：2024年4月26日*

[下载 (GitHub)](https://github.com/203-Systems/MatrixOS/releases/tag/V2.6.0)

新的 Matrix OS 带来了强大的新工具、优化和有趣的功能，进一步释放了 Mystrix 的潜力！

### 新功能
- **自定义控制面板应用**：超强大、可实时配置的自定义控制表面。使用 edit.203.io 自定义您的配置
- **和弦游乐场应用**：受 Poly Playground iOS 应用启发（演示版；在 Matrix OS 2.7 中合并到音符应用 V2）
- **拨弦应用 Beta** 🪕：使用 2 个触摸条将您的 Mystrix 当作 Omnichord 演奏
- **系统级 SysEx 和 HID API**：用于高级集成的 Beta 功能
- **自动音符关闭**：音符应用和表演应用操作菜单现在自动发送全部音符关闭信号 (CC 123)

### 改进
- **Mystrix 标准键盘驱动**：增强的响应性和准确性
- **通用设备优化**：更流畅、更快速的系统级性能

### 错误修复
- 修复了 Mystrix 触摸条切换状态未正确保存的问题
- 修复了翻转棋游戏应用的视觉更新
- 修复了翻转棋游戏应用的重置逻辑
- 修复了 Mystrix Pro pre-KC 批次设备配置未正确加载的问题

### 已知问题
- 自定义控制面板应用目前仅通过 USB MIDI 端口 1 发送 SysEx
- 如果 Mystrix 不是朝上或朝下放置，拨弦应用会崩溃

## Matrix OS 2.5 - Matrix OS 体验版
*发布日期：2023年10月25日*

[下载 (GitHub)](https://github.com/203-Systems/MatrixOS/releases/tag/V2.5)

添加了许多有用、炫酷或乱七八糟的功能到操作系统中。这是对 Matrix OS 和魔矩各种可能性的绝佳展示。

### 新功能
- **系统全局动画**：可切换的系统范围动画
- **Companion 应用集成**：BitFocus Companion 支持
- **游戏手柄应用**：可自定义键位映射功能的演示
- **骰子应用**：数字骰子滚动器
- **翻转棋游戏应用**：经典棋盘游戏实现
- **表演应用触控组合键**：增强的表演控制
- **更好的 LED 子系统**：支持抖动，实现更平滑的渐变
- **LED 分区**：Mystrix Pro 底灯的可调亮度倍数
- **双 MIDI 端口**：无需特殊驱动即可连接 Apollo Studio
- **增强的底灯**：启动动画和音符应用音符视图
- **切换按钮反馈**：在文本滚动中显示开/关状态
- **Mystrix Pro 键盘用户校准**：自定义灵敏度调整
- **音符应用垂直分割**：将键盘垂直分割为双区域
- **表演应用调色板**：4 个可自定义的调色板

### 更新
- **灯光应用 V2**：具有新功能的重大更新

### 改进
- **颜色选择器界面**：增强的用户界面
- **键盘驱动**：Mystrix Standard 和 Pro 的更好性能
- **LED 亮度步进**：更精细的控制
- **蓝牙安全连接**：Windows 和 Android 支持（通过 @M4kI9H7）

### 错误修复
- 修复了应用启动器中应用顺序错乱的问题
- 修复了内存泄漏
- 修复了表演应用闪烁减少切换
- 修复了音符应用重叠默认值显示
- 修复了发送特殊 MIDI 数据包的问题

### 更改
- **设备命名**：现在在设备名称中显示 Mystrix / Mystrix Pro
- **设备设置页面**：用于设备特定设置的新专用页面

### 移除
- **表演应用兼容模式**：Unipad 现在原生支持 Mystrix

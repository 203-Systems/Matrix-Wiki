# Poly Playground App

Poly Playground is a simple app inspired by the [Poly Playground App by Mike Gao](https://apps.apple.com/us/app/polyplayground/id511514938).

:::warning
It will be merged into Note App V2 in a future update.
:::

## Note Map

The note pad is chromatic, meaning every note is a semitone apart. On the X axis, each key is spaced by 3 semitones, and on the Y axis, each key is spaced by 4 semitones.

Pads with the same color use the same semitone (but may be in different octaves).

## MIDI Channel

This setting allows you to select the MIDI output channel (1-16) for the note pad.

# Poly Playground 应用

Poly Playground 是一个简单的小应用，灵感来自 [Mike Gao 的 Poly Playground 应用](https://apps.apple.com/us/app/polyplayground/id511514938)。

:::warning
未来更新中，它会合并进音符应用 V2。
:::

## 音符映射

音符面板是半音阶排列，也就是相邻音之间都是一个半音。X 轴方向每个按键相差 3 个半音，Y 轴方向每个按键相差 4 个半音。

同色的按键对应同一个半音（但可能在不同八度）。

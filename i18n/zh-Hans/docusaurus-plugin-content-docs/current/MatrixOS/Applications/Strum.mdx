# Strum 应用

Strum 是一个基于和弦的乐器应用，灵感来自 Omnichord 等设备。你可以选择和弦并使用魔矩设备左右两侧的触摸条来演奏。

:::note 需要设备旋转
Strum 应用需要将设备旋转为纵向模式（向左或向右旋转）以访问用于演奏的触摸条功能。
:::

## 使用方法

### 开始使用
1. **设备方向**：将设备旋转为纵向模式（向左或向右）以访问触摸条
2. **和弦选择**：点击底部行的任意和弦按钮来选择所需的和弦类型
3. **八度调节**：使用八度移调器（第2行）调整音高范围
4. **时长控制**：使用时长修改器（第3行）设置音符长度

### 弹奏
- **左侧触摸条**：在左侧触摸条上触摸并拖拽来弹奏和弦
- **右侧触摸条**：在右侧触摸条上触摸并拖拽来弹奏和弦
- **和弦可视化**：顶部区域显示所选和弦的音符和根音

### 可用的和弦类型
- **大三和弦**：基本大三和弦 (1-3-5)
- **小三和弦**：基本小三和弦 (1-♭3-5)
- **大七和弦**：带大七度的大三和弦 (1-3-5-7)
- **小七和弦**：带小七度的小三和弦 (1-♭3-5-♭7)
- **增三和弦**：增三和弦 (1-3-#5)
- **减三和弦**：减三和弦 (1-♭3-♭5)

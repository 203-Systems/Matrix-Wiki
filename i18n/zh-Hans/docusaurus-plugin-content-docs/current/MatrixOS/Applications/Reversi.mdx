# 黑白棋应用

黑白棋是一个双人策略棋盘游戏，在 8x8 的棋盘上进行。玩家轮流在棋盘上放置棋子，棋子朝上显示自己的颜色。在一次下棋中，任何被刚放置的棋子和当前玩家的另一颗棋子夹在一条直线上的对方棋子都会翻转为当前玩家的颜色。

在 [这里](https://en.wikipedia.org/wiki/Reversi) 了解更多关于黑白棋的信息。

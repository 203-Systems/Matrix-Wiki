# 黑白棋应用

黑白棋是一款双人策略棋盘游戏，在 8x8 的棋盘上进行。玩家轮流在棋盘上放置棋子，棋子正面显示自己的颜色。每次落子后，任何被新落子的棋子与当前玩家另一枚棋子在同一直线上夹住的对方棋子，都会翻转为当前玩家的颜色。

在 [这里](https://en.wikipedia.org/wiki/Reversi) 了解更多关于黑白棋的信息。

在游戏界面点击功能键即可打开配置菜单。

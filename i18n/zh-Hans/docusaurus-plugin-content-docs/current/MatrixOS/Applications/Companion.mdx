# Companion应用

Companion应用用于连接[Bitfocus Companion](https://bitfocus.io/companion/)，以实现对各种设备的高级和强大控制。

:::info
魔矩集成自[Bitfocus Companion](https://bitfocus.io/companion/) 3.4.0版本起已可用。
:::

# 音符应用

音符应用（V2）作为一个全面的MIDI音符界面，允许用户使用各种布局、音阶、调式和高级功能播放MIDI音符，包括自动和弦、琶音器、分屏界面和可自定义的颜色预设。

启动后，你将进入[音符板](#音符板)。点击功能键访问[配置菜单](#配置菜单)，再次点击返回[音符板](#音符板)。

要退出音符应用，请进入[配置菜单](#配置菜单)并按住功能键。这将使你返回到[应用启动器](/docs/MatrixOS/ApplicationLauncher)。

## 音符板

音符板是音符应用的主界面。它显示当前音符板，包含所选的布局、音阶和调式。

按键的外观会根据自定义选项而变化。请参阅[示例布局](#示例布局)以了解不同设置下的音符板外观。

点击功能键将带你进入[配置菜单](#配置菜单)。

## 布局选择器

布局选择器允许你在不同的音符板布局模式之间进行选择：

- **八度布局**：基于八度的布局。每一行都从根音开始。如果音阶有超过8个音符，将跨越2行。
- **偏移布局**：可自定义的偏移模式，具有可配置的X和Y偏移
- **半音布局**：传统的音符布局。音符按顺序排列，从左到右，从下到上
- **钢琴布局**：钢琴风格的按键排列

## 琶音器设置

琶音器设置菜单提供对琶音器设置的全面控制。此菜单仅在音符板界面中启用控制栏时可访问。

琶音器设置菜单有7个不同的参数页面，每个都有独特的颜色编码：

- **BPM**（红色）：控制琶音器的节拍/速度
- **摇摆**（橙色）：调整音符之间的时序摇摆
- **门限**（黄色）：设置音符持续时间/门限
- **顺序**（绿色）：选择琶音器模式顺序
- **步进**（青色）：控制琶音器前进的步数（每次步数偏移后循环#次）
- **步进偏移**（蓝色）：在每个步骤后添加音高偏移音符
- **重复**（紫色）：设置模式重复的次数

## 控制栏

控制栏在演奏视图中启用时提供对各种控制和效果的快速访问。

每个按钮也都有不同的Shift功能。按住任何八度按钮将启用Shift模式。

功能：
- **弯音向下** (X=1)：力度映射的弯音向下（Shift：MIDI停止）
- **弯音向上** (X=2)：力度映射的弯音向上（Shift：MIDI播放）
- **音符锁定** (X=3)：三种状态的音符延音（Shift：启用切换锁定模式）
- **和弦** (X=4)：打开和弦控制
- **琶音器** (X=5)：打开琶音器速度控制（Shift：进入琶音器设置）
- **调性模式** (X=6)：更改当前布局的调性（Shift：进入音阶选择器）
- **八度向下和向上** (X=7和8)：按八度移调音符（按住任一作为Shift）
- **交换布局**：同时点击八度向下+八度向上来交换布局。

**分屏模式下**：控制栏仅影响主要音符板（垂直分割中的左侧板，水平分割中的顶部板）。但是，您启用的任何模式或效果在切换音符板时都会保持。您可以在一个音符板上配置锁定、和弦或琶音器，然后切换到次要音符板，这些设置将继续独立运行。

:::note
**非持久性**：控制栏配置（锁定、和弦、琶音器等）不会跨会话保存。当您退出并重新启动音符应用时，所有控制栏设置都将重置为默认状态。
:::

**视觉反馈**：
  - **颜色**：主要音符板颜色
  - **渐变**：音符板八度

### 弯音

弯音控制允许您向上或向下弯曲演奏音符的音高。弯音量是力度映射的，这意味着按得更用力会产生更大的弯音。

**按钮**：
- **弯音向下** (X=0)：向下弯音（红色）
- **弯音向上** (X=1)：向上弯音（绿色）

**Shift功能**：
- **Shift + 弯音向下**：MIDI停止命令
- **Shift + 弯音向上**：MIDI播放命令

### 音符锁定

音符锁定允许您保持音符播放而不保持按键按下。该功能有三种模式。

**按钮**：
- **点击** (X=2)：在三种模式之间循环
- **Shift点击**：直接启用切换模式

**颜色状态**：
- **黄色**：锁定禁用 - 音符正常播放
- **白色**：锁定启用 - 按下的音符保持播放
- **洋红色**：切换模式 - 按键像开关一样工作

**模式行为**：
1. **禁用**（黄色）：标准音符播放
2. **启用**（白色）：所有按下的音符无限延续
3. **切换**（洋红色）：每个键在开/关状态之间切换

### 和弦控制

和弦控制允许您使用单个音符演奏完整的和弦。激活后，音符板顶部的两行变成和弦类型和转位的控制。

**按钮**：
- **点击** (X=3)：开关和弦控制
- **Shift点击**：锁定当前和弦选择

**颜色状态**：
- **青色**：和弦控制关闭
- **白色**：和弦控制开启

**和弦构建控制**：
- **和弦类型**：大和弦、小和弦、减和弦、增和弦 - 同时只能选一个
- **多个扩展**：6th、7th、9th - 您可以选择多个自由组合
- **按住组合**：在音符板上演奏时按住一个组合来使用和弦
- **Shift点击**：锁定和弦类型和扩展
- **转位**：选择一个转位或在锁定和弦时滑动穿过它们来扫弦。

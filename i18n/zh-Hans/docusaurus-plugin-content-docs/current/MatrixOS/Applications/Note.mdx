# 音符应用

音符应用是一个功能完整的MIDI音符界面，你可以用它来演奏MIDI音符，支持多种布局、音阶、调式，还有自动和弦、琶音器、分屏显示和颜色自定义等高级功能。

启动后会直接进入[演奏界面](#演奏界面)界面。点击功能键可以打开[配置菜单](#配置菜单)，再次点击即可返回[演奏界面](#演奏界面)。

要退出音符应用，先进入[配置菜单](#配置菜单)，然后按住功能键就能返回<DocLink to="/docs/MatrixOS/ApplicationLauncher">应用启动器</DocLink>。

## 演奏界面

演奏界面是音符应用的主界面，会根据你选择的布局、音阶和调式来显示不同的按键排列。

按键的颜色和样式可以自定义。想看看不同设置效果的话，可以查看[示例布局](#示例布局)。

点击功能键可以打开[配置菜单](#配置菜单)。

## 布局选择器

- **八度布局**：基于八度的布局，每一行都从根音开始，如果音阶包含超过 8 个音符，会跨越两行

- **半音布局**：传统音符布局，按半音顺序从左到右，从下到上排列
- **钢琴布局**：钢琴风格按键排列

## 琶音器设置

可全面控制琶音器设置，仅在演奏界面启用控制栏时可访问

包含8个不同的参数页面，每个都有独特的颜色标识：

- **BPM**（红色）：调节琶音器节拍速度
- **摇摆**（橙色）：调整音符时序摇摆
- **门限**（黄色）：设置音符持续时间
- **顺序**（绿色）：选择琶音模式顺序
- **步进**（青色）：设置琶音器步进数量
- **步进偏移**（蓝色）：在每步后添加音高偏移
- **重复**（紫色）：设置模式重复次数
- **欧几里得音序**（亮粉色）：欧几里得音序长度与步数

## 分屏模式

可选择启用或禁用分屏模式。分屏模式包括：无分屏、水平分屏或垂直分屏。

分屏模式选择器将根据当前的分屏模式显示不同的颜色：

- **无分屏模式**：暗白色
- **垂直分屏模式**：青色
- **水平分屏模式**：紫色

## 控制栏



功能：
- **弯音向下** (X=1)：力度映射的弯音向下（Shift：MIDI停止）
- **弯音向上** (X=2)：力度映射的弯音向上（Shift：MIDI播放）
- **音符锁定** (X=3)：三种状态的音符延音（Shift：启用切换锁定模式）
- **和弦** (X=4)：进入和弦控制
- **琶音器** (X=5)：进入琶音器速度控制（Shift：进入琶音器设置）
- **调性模式** (X=6)：更改当前布局的调性（Shift：进入音阶选择器）
- **八度向下和向上** (X=7和8)：按八度移调音符（按住任一作为Shift）
- **交换布局**：同时按下八度向下 + 八度向上来交换布局

**分屏模式下**：控制栏仅作用于主演奏界面（垂直分屏时为左侧板，水平分屏时为顶部板），你启用的模式或效果在切换演奏界面时会保持，你可以先在主演奏界面上配置锁定、和弦或琶音器，再切换到副演奏界面，这些设置会继续独立运行

:::note
**非持久性**：控制栏配置（锁定、和弦、琶音器等）不会跨会话保存，退出并重新启动音符应用时，所有控制栏设置都会重置为默认状态
:::

### 八度控制 / 移调

可按八度向上或向下移调所有音符。这些按钮也用作Shift修饰键来访问辅助功能。

**控制**：
- **八度向下** (X=7)：降低一个八度
- **八度向上** (X=8)：提高一个八度

**特殊功能**：
- **按住任意按钮**：启用其他控制的Shift模式
- **同时点击两个**：分屏模式下交换主次布局

**视觉反馈**：
  - **颜色**：主演奏界面颜色
  - **渐变**：演奏界面八度

### 弯音

可向上或向下弯曲演奏音符的音高。弯音量与力度映射，按得越用力弯音越大。

**按钮**：
- **弯音向下** (X=0)：音高向下弯曲（红色）
- **弯音向上** (X=1)：音高向上弯曲（绿色）

**Shift功能**：
- **Shift + 弯音向下**：MIDI停止命令
- **Shift + 弯音向上**：MIDI播放命令

### 音符锁定

可保持音符播放而无需持续按住按键。有三种模式。

**按钮**：
- **点击** (X=2)：在三种模式间切换
- **Shift点击**：直接进入切换模式

**颜色状态**：
- **黄色**：锁定禁用，音符正常播放
- **白色**：锁定启用，按下的音符保持播放
- **洋红色**：切换模式，按键像开关一样

**模式行为**：
1. **禁用**（黄色）：标准音符播放
2. **启用**（白色）：所有按下的音符持续播放
3. **切换**（洋红色）：每个键在开关状态间切换

### 和弦控制

可使用单个音符演奏完整和弦。激活后，演奏界面顶部两行变为和弦类型和转位控制。

**按钮**：
- **点击** (X=3)：切换和弦控制

**颜色状态**：
- **青色**：和弦控制已关闭
- **白色**：和弦控制已开启

**和弦构建控制**：
- **和弦类型**：大和弦、小和弦、减和弦、增和弦（同时只能选择一个）
- **扩展音**：六和弦扩展，小七和弦扩展，大七和弦扩展，九和弦扩展（可多选组合）
- **按住组合**：在演奏界面上演奏时按住一个组合来使用和弦
- **Shift点击**：锁定和弦类型和扩展
- **转位**：选择一个转位或在锁定和弦时滑动穿过它们来扫弦。

琶音器控制用于设置琶音器的分拍，让你在演奏时快速开启并调整琶音器速度

**按钮**：
- **点击** (X=5)：开关琶音器控制

**颜色状态**：
- **青柠色**：琶音器控制关闭

1. **关闭**

5. **1/12** (十二分音符)
6. **1/16** (十六分音符)
7. **1/24** (二十四分音符)
8. **1/32** (三十二分音符)

# 音序器应用

Matrix OS 的强大音序器应用。强烈建议先了解[音符应用](/docs/MatrixOS/Applications/Note)。

:::warning 预览版本
这里的文档尚不完整，可能存在事实错误。Matrix OS 3.2 仍未正式发布，系统内容也可能会变化。
:::

## 核心概念

- 轨道：独立的通道，拥有各自的片段、Pattern、颜色、通道和音符布局。本音序器共有 8 条轨道。 
- 片段：每条轨道的 Pattern 容器。片段可被启动或切换。本音序器每条轨道最多 14 个片段。
- Pattern：由步组成的网格。每个 Pattern 有独立的长度。本音序器每个片段包含 16 个 Pattern。
- 步：Pattern 中的时间片。一个步可包含多个事件。本音序器每个 Pattern 最多 16 步，但也支持微步。

### 轨道选择

- 点击轨道以选中它。选择不同轨道会清除步选择与正在发声的音符。
- 点击当前轨道可标记为“当前轨道已选中”（影响控制栏行为）。
- 清除 + 点击轨道会清空该轨道在当前片段中的所有步。

### Pattern 步进区

- 默认视图为 8x2（16 步）。双 Pattern 模式会扩展为 8x4（同时显示两个 Pattern）。
- 点按步以选中，松开即取消选中。
- 当音符垫上已选中音符时，点按步可在该步添加或移除这些音符。
- 点按步可试听其音符（Note On）；松开停止试听（Note Off）。
- 清除 + 点按步可清除该步中的事件。
- 复制模式：
  - 点按含事件的步以设置复制源。
  - 点按另一步将事件复制到该步（支持跨 Pattern 复制）。
- Shift + 点按含事件的步可打开步详情菜单（仅在该轨道未播放时）。

### 音符垫

- 用于输入与试听音符。
- 布局模式：Scale音阶视图、Chromatic半音视图、Piano钢琴视图、Drum鼓机视图。
- 在支持且启用时提供力度感应输入。
- 在音序器界面中，若已选择步，点按音符会在所选步上切换这些音符。
- 若音序器正在播放且录音已开启，输入的音符会直接被录入。

### Shift 修饰
当按住任一八度键时会进入此视图（按住时作为 Shift）。

- 按住 Shift：预览 Pattern 选择器
- 同时点按两个 Shift：切换是否启用 Pattern 选择器
- Shift + 步：[步详情菜单](#step-detail-menu)
- Shift + 播放（控制栏 x1）：继续播放
- Shift + 复制（控制栏 x6）：切换双 Pattern 视图

### Pattern 选择器

- 当 Pattern 选择器激活时显示（Shift 按住或锁定）。
- 点按 Pattern 槽以选择或切换 Pattern（播放时禁用切换）。
- 点按“添加”槽创建新的 Pattern。
- 清除 + 点按 Pattern 可删除。
- 复制模式：
  - 点按 Pattern 以设置复制源。
  - 点按另一 Pattern 进行复制。
- 按住 Pattern 进入长度调整模式，然后点按长度（1-16）设置该 Pattern 的长度。

## 片段启动器

用于选择片段播放的网格。类似 Ableton Live 的 Session 模式。

- 普通模式：横向 8 条轨道，纵向 7 个片段。片段分页选择第 1 或第 2 页。
- 宽片段模式：横向 4 条轨道，每条轨道占 2 列。

未播放时：
- 点按片段以选中。
- 点按空槽创建片段。
- Shift + 片段将仅播放该片段。
- 清除 + 片段会删除它。
- 复制模式：选择源片段，然后点按目标进行复制。

播放中：
- 点按片段将其排队。
- 点按空槽以排队停止该轨道。
- 双击片段可启动整行。

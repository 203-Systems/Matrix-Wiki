# 骰子应用

骰子应用是一个简单的应用程序，允许你掷骰子。

骰子有两种模式：点数骰子和数字骰子。

你可以自定义骰子的面数（点数模式为 2 到 9，数字模式为 2 到 99）以及动画的颜色和效果。

点击功能键可从骰子视图访问配置菜单。

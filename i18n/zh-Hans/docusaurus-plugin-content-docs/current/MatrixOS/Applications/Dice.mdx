# 骰子应用

骰子应用是个简单的小应用，用来掷骰子。

骰子有两种模式：点数模式和数字模式。

你可以设置骰子的面数（点数模式 2～9，数字模式 2～99），也能调动画的颜色和效果。

在骰子视图里按功能键，可以打开配置菜单。

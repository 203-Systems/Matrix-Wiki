# 灯光应用

灯光应用将你的设备转变为可以在各种环境中使用的多功能灯光面板。

在[配置菜单](#config-menu)中长按功能键将退出此应用并返回[应用程序启动器](/docs/MatrixOS/ApplicationLauncher)。

## 灯光视图

灯光视图是灯光应用的主界面。它显示当前的灯光效果和颜色。

点击功能键将带你进入[配置菜单](#config-menu)。

## RGB模式

在RGB模式下，你可以通过[选色器](/docs/MatrixOS/SharedUI/ColorPicker)将灯光视图的颜色设置为你想要的任何颜色。

## 色温模式

在色温模式下，你可以将灯光视图的颜色设置为代表色温的颜色。颜色将根据你设置的色温而变化。

选择器UI与[选色器](/docs/MatrixOS/SharedUI/ColorPicker)类似，但色相选择器被色温选择器取代，并且没有饱和度选择器。

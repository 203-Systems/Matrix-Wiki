# Matrix OS 基础

## 应用程序系统

Matrix OS 是魔矩及其他兼容设备的核心操作系统，将其转换为具备多模态连接和丰富独立功能的智能控制器。所有软件功能都作为独立的 Matrix OS 应用程序提供。

此文档仅涵盖原生应用程序。某些第三方应用程序可能有自己的手册。

## 启动

设备通电后会播放启动动画。如果 Matrix OS 无法连接到主机（例如仅连接移动电源），则会循环播放待机动画。此时可点击中心功能键进入操作系统。

启动动画结束后，[应用程序启动器](/docs/MatrixOS/ApplicationLauncher)将是操作系统显示的第一个界面。

## 用户界面

在大多数 Matrix OS 用户界面中，你可以长按任何亮起的按钮以查看其名称或功能。长按任何未亮起的按键将显示当前菜单名称。

某些亮起的按键在长按时会有更高级的控制面板（例如，[亮度控制](/docs/MatrixOS/SystemSettings#brightness-control)）。在这种情况下，你可以长按高级控制面板中的未亮起按键来查看每个按钮的功能。

## 退出应用

Matrix OS 提供两种退出应用程序并返回主屏幕的方法：

### 标准退出方法
1. **点击功能键（中心键）** 进入应用的操作/配置菜单
2. 在菜单中 **长按功能键** 退出应用

推荐使用此方法，因为它允许应用正确清理并保存必要数据。

### 紧急退出方法
如果在应用中卡住或需要强制退出：

**长按功能键 3 秒** 可立即返回主屏幕。

:::warning 紧急退出
3 秒强制退出在所有应用中都可系统级使用。当标准退出方法不可用或在你在应用界面中迷失时，可使用此方法。
:::

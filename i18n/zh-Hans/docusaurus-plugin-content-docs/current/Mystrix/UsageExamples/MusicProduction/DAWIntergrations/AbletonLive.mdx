# Ableton Live

:::info[功能开发中]
目前，魔矩**没有**通过 MIDI 远程脚本的原生 Ableton Live 集成。但是，这是我们正在积极探索的内容，**我们非常<DocLink to="/docs/Support/ConnectWithUs">欢迎你的反馈</DocLink>，告诉我们你希望在 Ableton Live 集成中看到哪些功能。**
:::

你仍然可以通过结合使用 Ableton Live 中的 MIDI 学习功能和 <DocLink to="/docs/MatrixOS/Applications/Performance">Performance 应用</DocLink>，将魔矩用作 Ableton Live 中的 MIDI 控制器。这使你可以将魔矩的控制映射到 Ableton Live 中的参数，或将其用作鼓垫。

此外，你可以在 <DocLink to="/docs/MatrixOS/Applications/Note">Note 应用</DocLink> 中使用魔矩，使你能够将魔矩用作可定制的音乐控制器。

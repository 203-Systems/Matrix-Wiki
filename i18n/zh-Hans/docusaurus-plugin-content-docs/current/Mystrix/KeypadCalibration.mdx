# 键盘校准

:::danger
此功能适用于高级用户，不建议普通用户使用。

键盘校准错误可能导致意外的设备行为。在最坏的情况下，你可能需要重新刷写操作系统并清除所有数据来恢复设备。
:::

:::info
此功能仅适用于具有压力感应键盘的魔矩设备，例如魔矩 Pro。
:::

该应用程序允许用户通过调整最小和最大触发力来根据自己的偏好校准键盘。

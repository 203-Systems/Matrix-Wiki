# 游戏手柄应用

游戏手柄应用是 Matrix OS 即将推出的可自定义键盘功能的简单演示。它将在未来版本中被可自定义键位映射应用程序取代。

:::tip
目前，我们为控制视图提供了固定布局，但你很快就能够通过 [Matrix OS 键位映射编辑器](https://edit.203.io) 构建自己的布局。
:::

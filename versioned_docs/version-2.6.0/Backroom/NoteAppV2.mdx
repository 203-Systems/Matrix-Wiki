# Note App V2

Note App V2 introduces new features including a control bar, arpeggiator, chord functionality, semitone color, and X offset.

The offset feature is primarily added to support Poly Playground-like note layouts.

:::info
Please note that this is a work in progress, and the design is subject to change. The current colors of the UI elements are for reference only, as our primary focus has been on functionality rather than visual design at this stage.
:::

## Arpeggiator Setting

You will be able to change the folowing settings of the arpeggiator:

- BPM: Set a BPM or sync to the external clock. (You can also send out a clock)
- Swing Amount: Set the swing amount of the arpeggiator. (-80% to 80%)
- Gate Time: Set the gate time of the arpeggiator. (0% to 100%)
- Direction: Set the direction of the arpeggiator. (Up, Down, Up/Down, Up/Down End Key Repeated, Random, Order, Chord)

- Step: Set the octave step of the arpeggiator. (1 to 8)
- Step Offset: Sets the offset for each step (-24 to 24)

- Follow Scale: Enable to follow the scale of the note pad layout. (Boolean)

# Color Picker

The Color Picker is a user interface for selecting a color. It is part of the shared UIs used in various applications.

The Color Picker is divided into two parts: Hue Selection and Shade Selection. First, you select the hue of the color you want to use, then you select the shade of the color.

If the function key is pressed inside the picker, the color picker will abort and return to the previous screen without changing the color.

# Lighting App

The Lighting App transforms your device into a versatile lighting panel that can be used in various environments.

Holding down the function key in the [config menu](#config-menu) will exit this app and bring you back to the <DocLink to="/docs/MatrixOS/ApplicationLauncher">application launcher</DocLink>.

## Lighting View

The Lighting View is the main interface of the Lighting App. It displays the current lighting effect and color.

Clicking the function key will bring you to the [config menu](#config-menu).

## RGB Mode

In RGB mode, you can set the color of the lighting view to any color you want via the <DocLink to="/docs/MatrixOS/SharedUI/ColorPicker">Color Picker</DocLink>.

## Temperature Mode

In Temperature mode, you can set the color of the lighting view to a color that represents a temperature. The color will change based on the temperature you set.

The selector UI is similar to the <DocLink to="/docs/MatrixOS/SharedUI/ColorPicker">Color Picker</DocLink>, but the hue selector is replaced with a temperature selector and there is no saturation selector.
